Q1. Filter Marks (90+)

We are given an array of marks of students. Filter out the marks of students that scored 90+. 

let marks = [85, 92, 78, 96, 88, 91];

let toppers = marks.filter((val) => {
    return val >= 90;
});

console.log(toppers);

Q2. Product using reduce()

Use the reduce() method to calculate the product of all numbers in the array. 

let numbers = [1, 2, 3, 4];

let product = numbers.reduce((acc, val) => {
    return acc * val;
}, 1);

console.log(product);

Q3. Sum from 1 to n

Take a number n as input from user. Create an array from 1 to n and calculate the sum using reduce(). 

let n = 5;

let arr = [];

for (let i = 1; i <= n; i++) {
    arr.push(i);
}

let sum = arr.reduce((acc, val) => {
    return acc + val;
}, 0);

console.log(sum);
