What is a Function?

Chapter 4 | Function in Js

A function is a block of code that performs a specific task and can be called whenever needed.

Example 1

function greet() {
  console.log("Hello World");
}

greet();

Output:

Hello World

Example 2 (With Parameters)

function sum(a, b) {
  console.log(a + b);
}

sum(5, 3);

Input: 5, 3

Output:

8

Function Definition

function functionName() {
  // do some work
}
function functionName(param1, param2) {
  // do some work
}

Function Call

functionName();

forEach Loop in Arrays(Higher Order Function /Method)

Syntax

arr.forEach(callbackFunction)

Callback Function

A callback function is a function that is passed as an argument to another function and executed later.

Explanation

  • forEach() is used to loop through an array
  • It executes a function for each element
  • It does not return a new array

Example

let arr = [1, 2, 3, 4, 5];

arr.forEach((val) => {
    console.log(val);
});
forEach Loop

Some More Array Methods - map()

Creates a new array with the results of some operation. The values its callback returns are used to form a new array.

arr.map( callbackFn(value, index, array) ) 

let arr = [1, 2, 3];

let newArr = arr.map((val) => {
    return val * 2;
});
map()

Some More Array Methods - filter()

Eg: all even elements

Creates a new array of elements that give true for a condition (filter).

arr.filter(callbackFn(value, index, array)) 

let newArr = arr.filter((val) => {
    return val % 2 === 0;
});
filter()

Some More Array Methods - reduce()

Performs some operations & reduces the array to a single value. It returns that single value. 

let arr = [1, 2, 3, 4];

let sum = arr.reduce((acc, val) => {
    return acc + val;
}, 0);
reduce()
