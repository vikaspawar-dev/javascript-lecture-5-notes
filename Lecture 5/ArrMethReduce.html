Some More Array Methods - reduce()

Performs some operations & reduces the array to a single value. It returns that single value.

arr.reduce(callbackFn(accumulator, currentValue))

Description

The reduce() method executes a callback function on each element and stores the result in an accumulator. Finally, it returns a single value.

Example (Sum of Array)

let arr = [1, 2, 3, 4];

let sum = arr.reduce((acc, val) => {
    return acc + val;
}, 0);

console.log(sum);

Explanation

acc = accumulator
val = current value

Step 1: acc = 0 + 1 = 1
Step 2: acc = 1 + 2 = 3
Step 3: acc = 3 + 3 = 6
Step 4: acc = 6 + 4 = 10

Final Output: 10
