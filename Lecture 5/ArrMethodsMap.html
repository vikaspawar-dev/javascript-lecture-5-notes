Some More Array Methods - map()

Definition

map() creates a new array with the results of some operation. The values returned by its callback function are used to form the new array.

Syntax

arr.map(callbackFn(value, index, array))

Example

let arr = [1, 2, 3, 4];

let newArr = arr.map((val) => {
    return val * 2;
});

console.log(newArr);

Key Points

  • Returns a new array
  • Does not change original array
  • Used for transformation
