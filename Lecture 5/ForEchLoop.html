forEach Loop in Arrays(Higher Order Function /Method)

Syntax

arr.forEach(callbackFunction)

Callback Function

A callback function is a function that is passed as an argument to another function and executed later.

Explanation

  • forEach() is used to loop through an array
  • It executes a function for each element
  • It does not return a new array

Example

let arr = [1, 2, 3, 4, 5];

arr.forEach((val) => {
    console.log(val);
});
    
 
Higher Order Function (HOF) – JavaScript
📌 Definition

Higher Order Function wo function hota hai jo:

kisi function ko argument (parameter) me leta hai
ya
kisi function ko return karta hai
🧠 Simple words me:

Function jo dusre function ke sath kaam kare = Higher Order Function

🔹 Example 1 (Function as Argument)
function greet(name) {
    return "Hello " + name;
}

function processUser(callback) {
    console.log(callback("Vikas"));
}

processUser(greet);

👉 Yaha greet ek function hai jo processUser me pass ho raha hai
👉 Isliye processUser = Higher Order Function

🔹 Example 2 (Function Returning Function)
function multiplier(x) {
    return function(y) {
        return x * y;
    };
}

let double = multiplier(2);
console.log(double(5)); // 10

👉 Yaha function ek aur function return kar raha hai
👉 Ye bhi Higher Order Function hai

🔹 Real Life Example (Array Methods)
let arr = [1, 2, 3];

arr.forEach((val) => {
    console.log(val);
});

👉 forEach() ek Higher Order Function hai
👉 Kyunki ye ek function (callback) ko accept karta hai

✅ Important Points
Callback function = function jo pass hota hai
HOF = function jo callback use karta hai
Common HOF:
forEach()
map()
