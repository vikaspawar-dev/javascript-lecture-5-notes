JavaScript - Functions

1.What is a Function?

A function is a block of code that performs a specific task and can be executed whenever required.

Example 1

function sayName() {
  console.log("My Name is Vikas");
}

sayName();

Output:

My Name is Vikas

Example 2 (With Parameters)

function multiply(x, y) {
  console.log(x * y);
}

multiply(4, 5);

Input: 4, 5

Output:

20

2.Function Definition

function functionName() {

function functionName(param1, param2) {
  // do some work
